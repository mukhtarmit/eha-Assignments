Starting My Web Development Journey

June 15th 2026

I recently started learning web development and programming. At first, HTML and CSS seemed challenging, but with consistent practice, I began to understand how websites are structured and styled.

One of the most exciting things I learned was how to create web pages using HTML elements such as headings, paragraphs, images, tables, and links. I also learned how CSS can transform a simple webpage into something visually appealing.

My goal is to continue improving my skills and eventually become confident in building complete websites and web applications.

Building My First Projects

June 17th 2026

After learning the basics of HTML and CSS, I started creating projects such as a travel webpage, recipe book, and personal blog. These projects helped me understand how different HTML elements work together.

The most rewarding part was seeing my work published through GitHub Pages. It felt great to share my projects online and view them as real websites.

Every project teaches me something new. I am becoming more comfortable with coding, debugging errors, and improving the appearance of my webpages.

I look forward to learning JavaScript and creating interactive websites in the future.
